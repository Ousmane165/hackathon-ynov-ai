# Phi-3.5-Financial

Ce dossier représente le modèle financier de production.

Dans un environnement réel, les poids du modèle pré-entraîné Phi-3.5-Financial sont placés ici ou téléchargés depuis la source validée.

Pour une démonstration locale reproductible, le modèle est instancié via Ollama :

```bash
ollama pull phi3.5
ollama create phi35-financial -f models/phi3_financial/Modelfile
```
