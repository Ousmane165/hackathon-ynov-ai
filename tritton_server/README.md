# Triton Inference Server

Ce dossier conserve l'alternative avancée demandée par le sujet.

La solution retenue pour la production est Ollama, mais la structure Triton est fournie pour montrer qu'un déploiement avancé reste possible.

Port HTTP Triton : `8000`.
Port gRPC Triton : `8001`.
Metrics : `8002`.
