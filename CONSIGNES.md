# CONSIGNES - Projet TechCorp

Objectif principal : déployer un modèle Phi-3.5-Financial accessible via une interface web de chat.

Objectif expérimental : préparer un fine-tuning médical LoRA à partir du dataset médical fourni.

Le rendu contient :

- serveur d'inférence Ollama ;
- interface web FastAPI/Jinja/JS ;
- configuration Triton conservée ;
- scripts de test ;
- rapport complet ;
- audit sécurité ;
- dossier medical_project pour le fine-tuning expérimental.
